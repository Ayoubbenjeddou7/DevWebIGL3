# Structure du Projet Angular

## src/app
Contient tous les composants, services, et modules de l'application. C'est le cœur de l'application.

## src/assets
Stocke les ressources statiques (images, fichiers, etc.) accessibles publiquement.

## src/environments
Contient les fichiers de configuration pour différents environnements (dev, prod).

## src/main.ts
Point d'entrée principal qui bootstrap l'application Angular.

## src/index.html
Page HTML principale qui charge l'application Angular.

## angular.json
Fichier de configuration pour Angular CLI (build, serve, test, etc.).

## package.json
Liste les dépendances npm et les scripts disponibles pour le projet.

## tsconfig.json
Configuration du compilateur TypeScript pour le projet.
