# Conclusion sur les Déclarations

## Pourquoi déclarer les composants?
- Angular doit connaître tous les composants avant de les utiliser
- La déclaration dans le module permet à Angular de compiler le composant
- Sans déclaration, Angular ne peut pas reconnaître le sélecteur

## Rôle du tableau declarations
- Liste tous les composants, directives et pipes du module
- Permet à Angular de les rendre disponibles dans les templates
- Un composant ne peut être déclaré que dans un seul module
